## Compile VIM automation
Just a simple shell script for automating VIM compiling
nothing more, nothing less. I wrote it for my self for when ever I need a fresh installation of vim
for any reason!

### Usage
Just clone it, give it permission and run it with sudo

```shell
git clone https://github.com/a-mashhoor/compile_vim.git compile_vim && cd compile_vim \
    && chmod u+x ./compile_vim.sh && sudo ./compile_vim.sh
```

happy coding :)
